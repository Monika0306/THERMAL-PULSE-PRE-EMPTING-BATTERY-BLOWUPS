# 🔋 Thermal Pulse: Pre-Empting Battery Blow Ups

An IoT-based battery monitoring system that helps prevent Lithium-ion battery overheating in Electric Vehicles by continuously monitoring battery health and providing real-time alerts.

## 🚀 Features
- 🌡 Monitors temperature and humidity
- ⚡ Measures battery voltage and current
- 🔔 Activates buzzer on overheating
- ☁ Uploads live sensor data to ThingSpeak
- 📊 Enables remote battery health monitoring

## 🛠 Tech Stack
- ESP32
- Arduino IDE
- DHT11 Sensor
- Current Sensor
- Voltage Sensor
- ThingSpeak Cloud
- Embedded C

## ⚙️ How It Works
1. ESP32 reads temperature, humidity, voltage, and current.
2. If the temperature exceeds the threshold, the buzzer is activated.
3. Sensor data is uploaded to ThingSpeak for real-time monitoring.
